What is this project?

This is a fan-made **PokeDex** web application built using HTML, CSS (with a focus on modularity), and pure JavaScript. The primary goal of this project is to showcase data retrieval, filtering, sorting, and user interface development using the public **PokeAPI**.

Key Features:

  • **Comprehensive Listing:** View all available Pokémon.
  • **Search & Filter:** Easily find Pokémon by name, ID, or Type.
  • **Generational View:** Browse Pokémon specific to each generation. The current generation is highlighted in the navigation bar.
  • **Detailed Information:** Click on any Pokémon card to view its stats, abilities, moves, and even hear its cry!
  • **Infinite Scroll:** Efficiently loads more Pokémon on the main page.
  • **Responsive Design:** Optimized for both desktop and mobile devices.

**Note:** All Pokémon data, images, and sounds are sourced from the open-source PokeAPI.
